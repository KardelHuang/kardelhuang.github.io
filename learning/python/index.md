## 🧠 學習亮點與對比（以 C# 為基準）

- **型別系統差異**：Python 是動態型別，你不必明確宣告型別，提升開發效率。
- **Async await 邏輯**：Python 也有 async/await（與 C# 類似），但 event loop 和 coroutine 概念略不同，值得深入探索。
- **錯誤處理**：Python 用 try/except/finally 模式，結構更為簡單。
